# My Portfolio Builder

A modern, interactive personal portfolio website showcasing AI & backend engineering expertise, production systems, and applied ML work. Built with React, TypeScript, Vite, and Tailwind CSS with smooth animations and a retro aesthetic.

## Features

- ✨ Smooth animations with Framer Motion
- 🎨 Beautiful UI with shadcn/ui components
- 📱 Fully responsive design
- ⚡ Fast build with Vite
- 🧪 Unit tests with Vitest
- 📝 TypeScript for type safety
- 🎯 SEO optimized
- 🌙 Dark mode ready

## Prerequisites

- Node.js 18+
- npm 9+ (or yarn/pnpm)

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/kparmar/my-portfolio-builder.git
cd my-portfolio-builder
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

The app runs on [http://localhost:8080](http://localhost:8080) by default.

## Available Scripts

- `npm run dev` - Start local development server with hot reload
- `npm run build` - Create optimized production build
- `npm run build:dev` - Create development build with source maps
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint to check code quality
- `npm run test` - Run test suite once
- `npm run test:watch` - Run tests in watch mode
